# 🏨 Taaza Resort - Professional Resort Website

## Overview
Taaza Resort is a fully functional, responsive website for a luxury resort. The website is built using HTML, CSS, and JavaScript with a modern design that works seamlessly across all devices.

## 📁 Project Structure
```
basic taaza/
├── index.html              # Homepage with hero section, about, and featured rooms
├── rooms.html              # Detailed information about all room types
├── amenities.html          # All resort amenities and facilities
├── booking.html            # Booking system, packages, and pricing
├── contact.html            # Contact information and inquiry form
├── css/
│   └── style.css          # Complete styling and responsive design
└── js/
    └── script.js          # Interactive functionality and validations
```

## 🚀 Quick Start

1. **Open in Browser**
   - Simply open `index.html` in any modern web browser
   - No server installation required

2. **File Structure**
   - Keep all files in the same directory
   - CSS files must be in `css/` folder
   - JavaScript files must be in `js/` folder

3. **Customization**
   - Edit text content directly in HTML files
   - Modify colors in `css/style.css` (`:root` variables)
   - Update contact information in pages
